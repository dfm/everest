The EVEREST catalog is available here and includes .fits files with the de-trended light curve data as well as .pdf data validation summaries associated with each EPIC target. Follow the links below for detailed information about the catalog. Note that it is highly recommended that users access the catalog through the everest interface, as this allows for post-processing of the light curves with custom masks.
