fits.py - FITS conversion¶

Converts the everest de-trended light curves to the FITS format using PyFits. These FITS files make up the public everest catalog.

everest.fits.MakeFITS(model, fitsfile=None)¶

Generate a FITS file for a given everest run.

Parameters:model – An everest model instance
