User Interface
==============

There are two ways of interacting with the :py:mod:`everest` catalog: via
the command line and through the Python interface. For quick visualization,
check out the command line tools below. For customized access to de-trended
light curves, including options to mask transits and refine the PLD model,
follow the link below to the Python code.
