Data Release Notes¶

  • Updates to the source code

    • 2.0.6 (02/20/2017)
  • Updates to the catalog

The everest source code and catalog are updated periodically as new K2 campaigns become available and as we make small changes to the de-trending algorithm. Below we keep a log of these changes. We encourage users to periodically upgrade their everest installation by running

pip install everest-pipeline --upgrade

Note that users must upgrade to enable access to new K2 campaigns.

Updates to the source code¶

2.0.8 (12/01/2017)¶

Updated catalog with K2 C102, C111, C112, and C13 light curves. Cleaned up the code to follow PEP-8 guidelines. Implemented overfitting metric calculations.

2.0.6 (02/20/2017)¶

Includes de-trended light curves from K2 campaigns 0-8. This is the first public release of the everest 2.0 code and is described in detail in our paper.
