everest-status - De-trending StatusÂ¶

The everest-status command prints the status of the batch de-trending runs for a given mission, model, and season. The command accepts several options, which we list below.

season
The season number. For K2, this is the campaign number. Note that
fractional seasons are allowed (i.e., 6.0). Default is 0
model
The everest model name. Default nPLD
-m mission
The mission name (k2 | kepler | tess).
Default k2
-s
Check the status of the short cadence runs.
-i
Check the status of the injection runs.
